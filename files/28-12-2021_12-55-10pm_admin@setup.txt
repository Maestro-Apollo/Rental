1. Install xampp in your laptop

2. After installing you will see xampp folder in your c drive. Go inside the folder and you will find htdocs folder.

3. Copy the zip file and paste it inside htdocs folder then unzip it.

4. After that open xampp control panel. You will find it by searching in start button. Once you open xampp control panel start apache and mysql.

5. After starting both of them, open browser and type localhost/phpmyadmin/

6. Click new on left side and create database called job_portal. After creating it you have to import the sql file.

7. Everything is ready now. Then open new tab and type in url section localhost/sproject/ And you will see the website working.
